# 📘 Binwalk Basic Scan
##🧩 Objective

To learn how to use Binwalk to scan a file and identify its internal structure.


## ⚙️ Step 1: Install Binwalk
```bash
sudo apt update
sudo apt install binwalk -y
```
✔️ This installs Binwalk on the system.


##⚙️ Step 2: Verify Installation
```
binwalk --version
```
✔️ Confirms that Binwalk is installed correctly.


## ⚙️ Step 3: Download Sample File
```
wget https://upload.wikimedia.org/wikipedia/commons/3/3f/JPEG_example_flower.jpg -O test.jpg
```
✔️ Downloads a sample image file for testing.


## ⚙️ Step 4: Scan the File
```
binwalk test.jpg
```
✔️ Scans the file and shows its internal structure.

### 📊 Output Observed
```
DECIMAL       HEXADECIMAL     DESCRIPTION
--------------------------------------------------------------------------------
0             0x0             JPEG image data, JFIF standard 1.01
```


## 🧠 What I Learned
- Binwalk scans files to detect internal data structures
- It identifies file types using known signatures
- A normal file (like a clean image) may not contain hidden data
- The output shows the file type and its starting position



## 💡 Key Concept
Binwalk works like an X-ray for files — it reveals what is inside a file without opening it normally.


## 🔑 Key Command
```
binwalk <filename>
```


## 📝 Conclusion

In this exercise, I successfully installed Binwalk and used it to scan a normal image file. The result showed that the file only contains standard JPEG data with no hidden content.
